# Tiny Form Designer Template

This template provides a minimal setup to Form Designer.

## Technologies

vite5 + react18 + ts + react-beautiful-dnd + @types/react-beautiful-dnd + ant-design

## For Developer

- pnpm i
- pnpm dev

> you should use pnpm@9 or higher
